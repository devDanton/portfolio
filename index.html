Sobre min

Sou um profissional dedicado e comprometido com a entrega do produto final. Nos últimos seis meses, busquei especializar-me ainda mais em diversas tecnologias front-end para atender a demanda. A partir disso, por meio da criação de projetos pessoais para portfólio com tecnologias como React, obtive imersão da importância da reutilização de código para diversas aplicações. Também utilizei TypeScript como principal linguagem de programação do back-end, essa salientou a importância da tipagem de dados para prevenção de erros. O framework Vite proporcionou para minhas aplicações web um excelente ecossistema para codificação e ótimo gerenciamento estrutural. Para modelagem do banco de dados destas aplicações utilizei o Prisma. Este é uma excelente alternativa para criação das tabelas, já que tem uma sintaxe mais acessível. Proporciona maior segurança dos tipos de preenchimento e o versionamento do banco de dados concede fácil acesso a versões anteriores. Através destas especializações, coloquei em prática diversos projetos que podem ser encontrados em meu GitHub
